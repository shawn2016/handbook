## 模板字符串

模板字符串是为了解决使用+号拼接字符串的不便利而出现的。它的功能非常强大，但是我们大多数时候使用它则非常简单。看一个例子大家就明白怎么使用了。

    // es6
    const a = 20;
    const b = 30;
    const string = `${a}+${b}=${a+b}`;

    // es5
    var a = 20;
    var b = 30;
    var string = a + "+" + b + "=" + (a + b);

使用 \`\` 将整个字符串包裹起来，而在其中使用 ${} 来包裹一个变量或者一个表达式。
